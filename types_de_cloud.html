Types de cloud

Tous les clouds ne sont pas identiques et aucun type de cloud computing ne convient à tout le monde. Plusieurs modèles, types et services différents ont évolué pour vous aider à trouver la solution adaptée à vos besoins.

Vous devez commencer par déterminer le type de déploiement cloud ou d’architecture de cloudcomputing sur lequel vos services cloud seront implémentés. Il existe trois modes de déploiement de services cloud : le cloud public, le cloud privé et le cloud hybride.
